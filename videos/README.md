# Vídeos do Spring Space

Coloque os arquivos de vídeo (`.mp4`) nesta pasta e registre-os na lista `VIDEOS`
em `js/script.js`. Cada item segue este formato:

```js
{
    file: 'celebrando.mp4',       // arquivo dentro de /videos
    poster: 'evento6/3.jpg',           // imagem de capa (pode ser de qualquer evento)
    title: 'Celebração',    // título exibido na galeria
    category: 'Celebração'              // casamento | aniversario | boda | video
}
```

Depois de salvar, os vídeos aparecem automaticamente na galeria e podem ser
abertos em tela cheia ao clicar, inclusive pelo filtro **Vídeos**.
